
# ⭐ Consolidated Cheat Sheet: Microservice → DB Choice Logic

## ✔ Use **SQL** for:

-   Transactions
    
-   Workflow state
    
-   Referential master data (vendor, PO, ledger)
    
-   Compliance
    
-   Anything requiring ACID
    

## ✔ Use **MongoDB / Document DB** for:

-   Flexible product catalogs
    
-   Reviews, user-generated content
    
-   JSON-heavy dynamic objects
    

## ✔ Use **Redis** for:

-   Caching
    
-   Sessions
    
-   Cart
    
-   Balance snapshot
    
-   Rate limiting
    

## ✔ Use **ElasticSearch** for:

-   Fuzzy search
    
-   Filters + ranking
    
-   Query autocomplete
    

## ✔ Use **Cassandra / Wide Column** for:

-   High-write scenarios (inventory, event logs)
    

## ✔ Use **Neo4j (Graph)** for:

-   Fraud detection
    
-   Approval hierarchies
    
-   Recommendation graphs
    

## ✔ Use **Object Storage** for:

-   PDFs, images, invoices, receipts, user uploads
    

## ✔ Use **Kafka** for:

-   Real-time events
    
-   Immutable audit trails
    
-   Event-driven workflows
